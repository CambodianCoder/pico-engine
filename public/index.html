Reset database



If you wish to start over,
remove the database.
First, stop your
HTTP server (use Ctrl-C).
Use rm -r db
to clear the database.
Then restart the HTTP server
(use npm start)
which will create an empty
database.
Refresh this Bootstrap page
each time you
clear the database.
